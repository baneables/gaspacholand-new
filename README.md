# GaspachoLand

A beautiful website for GaspachoLand, a private Minecraft roleplay server with mods.

## Features

- **Home Page**: Welcome message with animations and call-to-action buttons
- **Rules Page**: Comprehensive server rules
- **Mods Page**: List of mods used on the server
- **Participants Page**: Information about the community and staff
- **Announcements Page**: Latest server news and updates
- **Application Form**: Functional form for new players to apply
- **Dark/Light Mode Toggle**: Switch between themes
- **Responsive Design**: Works on all devices
- **Animations**: Smooth animations using Framer Motion

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Technologies Used

- Next.js 16
- TypeScript
- Tailwind CSS
- Framer Motion
- Next Themes
- Lucide React
